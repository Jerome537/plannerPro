# 🏗️ PlannerPro - Créateur de Plans Extérieurs

Application web intuitive pour concevoir des plans d'aménagement extérieur professionnels.

## ✨ Fonctionnalités

- 📦 **Bibliothèque d'objets** riche et variée
- 📏 **Dimensions précises** en mètres avec calcul des surfaces
- 🎨 **Personnalisation** complète (couleurs, rotation, noms)
- 📚 **Système de calques** pour organiser votre plan
- 💾 **Sauvegarde/Chargement** de projets (JSON)
- 🖼️ **Export** en image PNG haute qualité
- 📊 **Statistiques** en temps réel

## 🚀 Installation

Visitez directement : https://jerome537.github.io/plannerPro/

Ou clonez le repository :
